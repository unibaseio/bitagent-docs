# FAQ

* **What is the agent creation fee?** \
  The agent creation requires a service fee of 0.2 BNB.
* **What is the transaction fee?** \
  A 1% fee is charged on buying and selling agent tokens, which is used to support platform operations and ecosystem incentives.
* **What is knowledge mining?** \
  Users contribute specialized knowledge that fits the agent’s needs, helping to evolve the agent, and earn token rewards based on contribution value.
* **How does an agent graduate?** \
  When the agent's market value within the bonding curve reaches 100 BNB, it "graduates" and the trading pair is created on PancakeSwap, with liquidity injected (after deducting 1 BNB service fee).
* **What image formats are supported?** \
  Common formats like .jpg, .png, .gif are supported.
* **How are contributions tracked?** \
  All data contributions are recorded and verified using contribution scores to ensure transparent and fair reward distribution.
* **What is the Hypersynchronicity mechanism?** \
  A parallel processing mechanism that ensures agents can efficiently respond to multi-channel instructions even under high concurrency conditions.
