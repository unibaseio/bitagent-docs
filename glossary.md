# Glossary

* **Agent:** A decentralized AI entity with on-chain identity, exclusive wallet, and permanent memory, capable of autonomous decision-making and task execution.
* **SubDAO:** A sub-decentralized autonomous organization used for local governance of the agent ecosystem, enabling community participation in decision-making.
* **Multimodal Interaction:** The ability to process multiple input and output types, such as text, voice, and images, enhancing interaction experience.
* **Smart Contract:** A program deployed on the blockchain that automatically executes predefined logic, ensuring transparency and fairness in interactions and incentives.
* **AIP Protocol:** A protocol that supports interoperability between agents, facilitating cross-platform data sharing and collaborative memory building.
* **Hypersynchronicity:** A parallel processing mechanism designed to improve system efficiency under high-concurrency conditions.
* **Bonding Curve:** A mechanism that dynamically adjusts the price of the agent's token during the incubation phase, reflecting changes in market demand.
