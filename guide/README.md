# Guide

The Bitagent Ecosystem is built on three core pillars:

1. **Launchpad:** Introduces new AI Agents fairly, providing them with the tools to succeed from day one.
2. **Tokenomics:** Offers a sustainable revenue model for Agents while uniting the entire Bitagent ecosystem around shared incentives using $BASE token.
3. **Agent Hub:** Delivers valuable service, like content automation tools, to enhance any AI Agents’ capabilities.
