# AI Agent Creation

Bitagent has integrated the latest large language models (LLMs), such as OpenAI, into its backend infrastructure. You can launch your own agent with a simple configuration on the Bitagent creation page.

#### \*Step 1. Set the token information details. <a href="#step-1.-set-the-token-information-details" id="step-1.-set-the-token-information-details"></a>

* Avatar
* Agent name
* Ticker
* Introduction

#### \*Step 2. Bonding curve design

* Choose parent token, currently supports USD1, WBNB
* (Optional) DEV Reserve
