# Bonding Curve

The Bonding Curve Launch is an open, sniper-resistant sale format where token prices increase with each buy. Once liquidity goals are reached, Agent Keys are automatically deployed to PancakeSwap (BNB), enabling smooth post-launch trading.

\
**Launch Mechanism**

* "Pump-fun" style
* Sale Allocation: 85% sold via bonding curve
* Liquidity to DEX: 15%
* Creation Fee: 0.02 BNB
* Trading Fee: 1% on Bonding Curve, 0.25% on DEX
