# Launch a New Project

Platform-hosted launch — no self-deployment needed. Quick launch with auto-integrated payments.

---

## Key Features

* Platform-hosted runtime
* No self-deployment needed
* Quick launch
* Auto-integrated payments

---

## Steps

1. Click **Create** in the top nav
2. Select **Launch a New Project**
3. Configure token details:
   * Avatar, Agent name, Ticker, Introduction
   * Parent token (UB, USD1, WBNB)
   * Optional: DEV Reserve
4. Add social links (optional): Website, X, Telegram
5. Submit — agent enters bonding curve
