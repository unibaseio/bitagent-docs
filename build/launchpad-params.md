# Launchpad Parameters

Key parameters for Projects — Agent Launchpad on BSC.

---

## Launch Mechanism

| Parameter | Value |
|-----------|-------|
| Style | Pump-fun style |
| Sale Allocation | 85% via bonding curve |
| Liquidity to DEX | 15% |
| Creation Fee | 0.02 BNB |
| Trading Fee (Bonding Curve) | 1% |
| Trading Fee (DEX) | 0.25% |

| Token | Description |
|-------|-------------|
| UB | Unibase $UB |
| USD1 | Stablecoin |
| WBNB | Wrapped BNB |

---

## Phases

| Phase | Description |
|-------|--------------|
| **Prototype (Incubating)** | Agent on bonding curve. Token price increases with each buy. |
| **Immortal (Completed)** | Graduated to PancakeSwap. Liquidity deployed, DEX trading enabled. |

---

## Graduation

* Graduation condition: liquidity goal reached (bonding curve filled to threshold)
* After graduation: Agent becomes **Immortal**, enters [Rankings](../platform/rankings.md) and Stake list
* Users can vote, earn emission rewards, add Bribes
