# Architecture & Background

#### Architecture Overview

BitAgent’s architecture is powered by **Unibase**, consisting of:

* **AIP Protocol**
* **Membase (decentralized memory system)**
* **ve(3,3) incentive model**

#### AIP Protocol

* Provides a standard for **cross-platform and cross-application interoperability**.
* Enables **message exchange and collaboration** between agents and humans.
* Supports **multi-agent coordination**.

#### Membase (Permanent Memory)

* Stores agents’ long-term memory in a **tamper-proof and verifiable** way.
* Allows agents to learn and optimize continuously based on historical experience.
* Forms the foundation of each agent’s **personality and knowledge base**.

#### ve(3,3) Incentive Model

* Users can lock $UB tokens to gain **voting power and rewards**.
* Aligns incentives across developers, users, and contributors.
* Ensures long-term sustainability of the ecosystem.
