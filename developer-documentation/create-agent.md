# Create Agent

Bitagent offers a zero-code deployment feature, greatly lowering the entry barrier for developers. The steps are as follows:

1. **Wallet Connection:** Link your wallet through the top-right link on the platform page, ensuring the wallet balance is sufficient (greater than 0.2 BNB).
2. **Agent Information Configuration:**&#x20;
   * Provide agent image (supported formats: .jpg, .png, .gif), agent name, token ticker, and a brief description.
   * Set the allocation ratio between DEV Reserve and Bonding Curve. (Developers can determine the proportion of DEV Reserve based on the project's actual needs and must clearly explain its usage to the community. This portion will be directly allocated to the DEV wallet. The setting range covers 80% of the total token supply, excluding the 20% allocated for the liquidity pool (LP). A reasonable reserve ratio and transparent usage explanation are critical to the success of the project. **Please note: Once set, it cannot be modified.**)
   * Optionally, you can add social links such as Twitter (X), Telegram, or the official website
3. **Pay the Service Fee and Create:** Click the "Create" button to submit the request. After paying the 0.2 BNB service fee, the system will automatically deploy the agent and generate the agent’s token.
4. **Manage Agent:** After creation, the agent’s detail page will be accessible. Users can directly buy or sell agent tokens and participate in knowledge mining to help evolve the agent.
