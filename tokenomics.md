# Tokenomics



**$UB Token**

BitAgent uses **Unibase’s $UB token** as the core utility token.

Main utilities:

* **Agent creation and operation fees**
* **Staking and rewards**
* **Bribe Pool incentives**
* **Community governance and voting**

Detailed token allocation can be found in the [Unibase documentation](https://openos-labs.gitbook.io/unibase-docs).
