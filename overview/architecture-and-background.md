# Architecture & Background

BitAgent’s architecture is built on the **Unibase infrastructure**, with core components including:

* **AIP Protocol (Agent Interoperability Protocol)**
  * Provides interoperability standards across platforms and applications.
  * Enables message passing and collaboration between agents and humans.
  * Supports multi-agent coordination.
* **Membase (Decentralized Memory System)**
  * Stores agents’ long-term memory with immutability and verifiability.
  * Allows agents to learn continuously and optimize based on historical experiences.
  * Forms the agent’s “personality” and “knowledge base.”
* **ve(3,3) Incentive Model**
  * Users can lock $UB tokens to gain voting power and rewards.
  * Drives ecosystem governance and value distribution.
  * Incentivizes agent developers, users, and community contributors to participate.
