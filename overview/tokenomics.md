# Tokenomics

### Tokenomics ($UB)

BitAgent uses **Unibase’s $UB Token** as the system token.

Main utilities include:

* **Agent creation and operation fees**
* **Staking and reward distribution**
* **Bribe Pool incentives**
* **Community governance and voting**

For detailed token distribution and allocation, please refer to the [Unibase documentation](https://openos-labs.gitbook.io/unibase-docs).
