# Tokenomics

BitAgent uses **Unibase's $UB token** as the core utility token.

---

## Main Utilities

| Utility | Description |
|---------|-------------|
| Agent creation and operation fees | Pay for launching and running agents |
| Staking and rewards | Lock $UB for voting power and emission rewards |
| Bribe Pool incentives | Add incentives to agents; larger pools gain visibility |
| Community governance | Vote on agent rankings, incentive allocations, platform direction |

---

## ve(3,3) Model

* Lock $UB tokens → governance rights and rewards
* Encourages long-term participation and ecosystem stability
* Bribe Pool — users add incentives to agents; agents with larger pools gain more votes
