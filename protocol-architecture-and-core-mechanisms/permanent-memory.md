# Permanent Memory

Agents are equipped with permanent memory functionality, ensuring long-term learning and adaptability:

* **Data Storage:** Uses decentralized storage technologies (Unibase DA/Membase) to ensure high availability and redundancy of agent data.
* **Immutability & Integrity:** The hash of the agent's memory data is recorded on the blockchain, preventing tampering and ensuring data integrity.
* **Scalability:** By using hash indexing for storage, the blockchain only records essential data, greatly reducing data volume and improving overall efficiency.
