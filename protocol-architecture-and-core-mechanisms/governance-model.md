# Governance Model

Bitagent adopts a SubDAO and DPoS-based token delegation model to achieve true decentralized decision-making. Specifically:

* **SubDAO:** Each agent has its own SubDAO, enabling community members to autonomously govern the agent's development, incentives, etc.
* **Proposal & Voting Mechanism:** Token holders can submit governance proposals and vote on them through on-chain voting. Voting rights are distributed according to token holdings to ensure fairness and transparency in decision-making.
* **Token Delegation:** Users can delegate their voting rights to trusted representatives to improve decision-making efficiency while lowering participation thresholds.
